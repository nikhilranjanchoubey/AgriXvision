# AgriXvision - AI-Powered Crop Health Monitoring

A fully responsive, modern website for **AgriXvision**, an AI-powered agricultural monitoring system that helps farmers detect crop diseases, predict soil conditions, and monitor pest risks using advanced multispectral imaging and sensor data.

## üå± Project Overview

**AgriXvision** is a comprehensive agricultural technology solution that leverages artificial intelligence to revolutionize farming practices. Our platform provides real-time insights for disease detection, soil quality assessment, and pest risk prediction, empowering farmers with data-driven decision making.

### Key Features
- **Crop Disease Detection**: Early identification using computer vision and ML algorithms
- **Soil Quality Prediction**: Advanced analysis for optimal fertilization strategies
- **Pest Risk Monitoring**: Proactive detection to prevent crop damage
- **Multilingual Support**: Farmer-friendly interface in English and Hindi
- **Mobile-First Design**: Responsive across all devices

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. For development, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```
